# OmniVPNClient 🚀

A production-grade macOS VPN client built using Network Extension framework with Clean Architecture and Domain-Driven Design.

## 🧠 Architecture

* Clean Architecture (Domain / Application / Infrastructure)
* MVVM for UI
* Protocol-oriented design
* Plugin-based extensibility

## 🔐 Features

* Packet Tunnel VPN (NEPacketTunnelProvider)
* DNS Proxy support
* MDM configuration
* Policy-based filtering
* Observability (logging + metrics)

## 🧱 Tech Stack

* Swift / SwiftUI
* NetworkExtension
* Combine / Async-Await
* Swift Package Manager

## 📊 System Design

* Packet processing pipeline
* Policy engine
* Secure tunnel routing
* Modular domain architecture

## 🚀 Getting Started

1. Open in Xcode
2. Enable Network Extension capability
3. Configure App Group
4. Run main app target

## ⚠️ Notes

* Requires Apple Network Extension entitlement
* Demo project for enterprise VPN architecture
